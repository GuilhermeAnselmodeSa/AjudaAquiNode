Tutorial NodeJS

Olá, todo este projeto serve de instrução para o aprendizado na construção de aplicações utilizando o NodeJs

O arquivo app.js contém o básico, comece por la, com ele você aprende a abrir um servidor e exibir uma mensagem

O arquivo index.js contém como utilizar o express, o nodemon, criar rotas e utilizar parâmetros

Este arquivo se trata de uma instrução básica de como utilizar NodeJS+HTML, para assim, em projetos futuros, estar aplicando os conhecimentos. Válido citar que partes deste projeto estara no index.js
